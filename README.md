# List Maker Webapp

## Description

A simple Todo/Shopping list handling webapp with CRUD functions

## Technologies used:

- ASP.NET Web API
- Models & Controllers
- Ajax
- HTTP Requests
- Status Codes
- HTML & CSS
- jQuery & JavaScript
- DOM manipulations
- Entity Framework
- Object relational mapping (ORM)
- DataContext
- Database
